Guía para clientes

Guía para crear las cuentas que necesitarás para tu página web

Para que tu página web funcione correctamente, algunas herramientas deben quedar registradas directamente a tu nombre. Esto te permite mantener control total sobre tu dominio, tu correo profesional y cualquier integración que utilice tu proyecto.

Descargar PDF Ver otros recursos

En esta guía encontrarás las plataformas que más recomendamos y con las que trabajamos habitualmente. Sin embargo, no estás obligado a utilizar ninguna de ellas en particular. Si ya utilizas otro proveedor o prefieres una alternativa diferente, cuéntanos y te ayudaremos a configurarla de la misma forma. No necesitas crear todas las cuentas que aparecen aquí, solo aquellas que correspondan a tu plan y a las funcionalidades que hayamos definido juntos.

Importante: Durante la configuración inicial es posible que te solicitemos acceso temporal a algunas de estas cuentas para conectar correctamente tu proyecto. Una vez finalizado el proceso, te recomendamos cambiar tu contraseña por seguridad.

1. GitHub (aquí vive tu página web)

GitHub es donde se guarda el código de tu página y desde donde se publica gratuitamente en internet mediante GitHub Pages.

No necesitas conocimientos de programación para utilizar GitHub. Nosotros nos encargaremos de toda la parte técnica necesaria para publicar tu página.

Pasos

  1. Ve a github.com.
  2. Haz clic en Sign up.
  3. Ingresa tu correo electrónico, crea una contraseña segura y elige un nombre de usuario.
  4. Verifica tu correo con el código que te enviarán.
  5. Selecciona el plan Free cuando te lo pregunten.
  6. Avísanos cuando tengas tu cuenta lista para conectar tu proyecto.

2. Vercel (para pagos en línea y funciones avanzadas)

Utilizamos Vercel cuando una página requiere funcionalidades más avanzadas, como procesamiento de pagos o integraciones que necesitan una capa adicional de seguridad.

Pasos

  1. Ve a vercel.com.
  2. Haz clic en Sign Up.
  3. Puedes registrarte con tu cuenta de GitHub (recomendado) o con tu correo electrónico.
  4. Selecciona el plan Hobby (gratuito), salvo que te indiquemos lo contrario.
  5. Avísanos cuando tengas tu cuenta lista para realizar la conexión.

3. Porkbun (para comprar y administrar tu dominio)

Porkbun es donde registrarás el nombre de tu página web (por ejemplo, tuconsultorio.com) y desde donde lo conectarás con el hosting.

Pasos

  1. Ve a porkbun.com.
  2. Busca el dominio que deseas utilizar.
  3. Si está disponible, agrégalo al carrito.
  4. Crea tu cuenta antes de realizar la compra.
  5. Completa tus datos y verifica tu correo.
  6. Finaliza el proceso de compra.
  7. Avísanos cuando lo tengas listo para hacer la configuración.
Si aún no sabes qué dominio elegir, podemos ayudarte a encontrar una opción adecuada para tu proyecto.
Cada tipo de dominio tiene su propio tiempo de renovación. Te recomendamos revisar la fecha de vencimiento en tu panel de Porkbun y activar la renovación automática para no perderlo.

4. Calendly (para que tus pacientes agenden citas)

Calendly permite que tus pacientes soliciten citas directamente desde tu página web según tu disponibilidad.

Pasos

  1. Ve a calendly.com.
  2. Haz clic en Sign up free.
  3. Regístrate con tu correo o cuenta de Google.
  4. Configura tu nombre y zona horaria (Bogotá, Colombia).
  5. Define los días y horarios en los que atiendes.
  6. Copia el enlace de tu calendario y compártelo con nosotros.

5. Formspree (para recibir los mensajes del formulario de contacto)

Formspree permite que los mensajes enviados desde tu página lleguen directamente a tu correo electrónico.

Pasos

  1. Ve a formspree.io.
  2. Haz clic en Get Started.
  3. Regístrate con tu correo o cuenta de Google/GitHub.
  4. Confirma tu correo electrónico.
  5. Crea un nuevo formulario.
  6. Copia el ID que genera la plataforma.
  7. Compártenos ese ID para conectarlo con tu página.

6. Google Workspace (recomendado para tu correo profesional)

Google Workspace te permite utilizar un correo con tu propio dominio, por ejemplo: hola@tuconsultorio.com. Aunque no es obligatorio, recomendamos utilizar un correo profesional con dominio propio porque transmite una imagen más confiable y profesional.

Pasos

  1. Ve a workspace.google.com.
  2. Haz clic en Empezar.
  3. Ingresa el nombre de tu negocio.
  4. Cuando te pidan un dominio, selecciona que ya tienes uno.
  5. Sigue el proceso de verificación.
  6. Avísanos cuando termines para ayudarte con la configuración necesaria.

7. Anthropic / API de Claude (solo para Chatbot Pro)

Esta es la plataforma de inteligencia artificial que permite que tu chatbot genere respuestas inteligentes para tus pacientes. Este servicio tiene costos independientes definidos por Anthropic según el uso real que tenga tu chatbot.

Pasos

  1. Ve a console.anthropic.com.
  2. Haz clic en Sign Up.
  3. Regístrate con tu correo electrónico.
  4. Verifica tu correo.
  5. Ve a Billing y agrega tu método de pago.
  6. Ve a API Keys y genera una nueva clave.
  7. Compártenos la clave de forma segura para realizar la integración.
Importante: Esta clave funciona de forma similar a una contraseña. Una vez terminemos la integración, te recomendamos generar una nueva clave y eliminar la anterior por seguridad.

¿Necesito crear todas estas cuentas?

No. Cada proyecto es diferente y no todos los planes utilizan las mismas herramientas. Por ejemplo:

  • Un Sitio Esencial normalmente requiere GitHub y dominio.
  • Un Sitio Profesional puede incluir además Calendly o Formspree.
  • Un Sitio con Chatbot puede requerir Anthropic y otras integraciones adicionales.

Antes de comenzar, te indicaremos cuáles necesitas crear para tu caso.

Si durante el proceso encuentras alguna diferencia respecto a lo que aparece en esta guía, no te preocupes. Las plataformas pueden cambiar su diseño o sus opciones con el tiempo. Si tienes dudas en cualquier paso, estaremos encantados de ayudarte para que todo quede correctamente configurado.

Tu página web es tuya. Estas cuentas también deberían serlo. Por eso recomendamos que siempre queden registradas directamente a tu nombre y bajo tu control.
